# 💬 Nivesh Mitra – Multilingual FD Advisor

🚀 An AI-powered multilingual chatbot that simplifies complex financial terms like Fixed Deposits (FD) into easy, beginner-friendly language.

## 💡 Problem Statement

Many first-time investors in India struggle to understand financial products due to:
- Complex jargon (interest rate, tenure, etc.)
- Language barriers
- Lack of simple explanations

---

## ✅ Solution

**Nivesh Mitra** solves this by:
- Explaining financial concepts in **simple language**
- Supporting **multiple Indian languages**
- Providing a **chat-based interactive experience**

---

## ✨ Features

- 🌐 Multilingual chatbot (Hindi, English, Bengali, Tamil)
- 🧠 Simplifies financial jargon into easy explanations
- 💬 Conversational UI (chat-based interface)
- 🎯 Beginner-friendly responses
- ⚡ Fast AI responses using Groq API

---

## 🧠 How It Works

1. User opens chatbot
2. Selects preferred language
3. Asks a financial question (e.g., “What is FD?”)
4. AI responds in simple, easy-to-understand language

---

## 🛠️ Tech Stack

### Frontend
- React (Vite)
- CSS

### Backend
- Node.js
- Express

### AI
- Groq API (LLM for generating responses)

## ⚙️ Setup Instructions

### 🔹 Clone Repository


git clone [https://github.com/yourusername/nivesh-mitra.git](https://github.com/mayankjain2710/nivesh-mitra)

cd nivesh-mitra


---

### 🔹 Backend Setup


cd Backend
npm install
node server.js


---

### 🔹 Frontend Setup


cd frontend
npm install
npm run dev


---

## 🔐 Environment Variables

Create a `.env` file in the Backend folder:


GROQ_API_KEY=your_api_key_here
